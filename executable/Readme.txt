Pour lancer le programme en ligne de commande:

java -jar GribouilleGame.jar [WIDTH] [HEIGHT] [POINT_NUMBER] [MAX_SHAPES]

Si vous ne précisez pas de paramètres alors le programme se lancera avec les valeurs par défault:

WIDTH = 1600 - La largeur de la fenetre
Pour un affichage complet du menu il est préférable de ne pas descendre sous 1200

HEIGHT = 800 - La hauteur de la fenetre
POINT_NUMBER = 40 - Le nombre de points à recouvrir
MAX_SHAPES = 4 - Le nombre maximum de forme que l'on peut placer

Sous windows, vous pouvez aussi double cliquer sur le .jar
